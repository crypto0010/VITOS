VITOS

VIT Operating System

A Kali-based academic security distribution with AI-powered behavioral monitoring, sandboxed pentesting tools, kernel-level eBPF telemetry, WireGuard/Tor Ghost Mode, and a real-time admin dashboard. Designed and developed at the Cybersecurity and Digital Forensics Lab, VIT Bhopal University.

About

Built for the academic cybersecurity classroom

VITOS is a complete operating system that pairs the standard Kali Linux pentesting toolchain with an academic-integrity layer: every student action is sandboxed, every tool execution is logged, and a local AI engine watches for behavior outside the assigned lab exercise. It's the operating system our cybersecurity students actually use — built by us, for us.

v1.0.0 ships 8 Debian packages on top of a custom hardened Linux 6.6 kernel, fits on a 16 GB USB stick, and boots on any x86-64 lab workstation. The whole stack runs locally — no cloud dependency, no telemetry leaving the lab network.

Each layer is a Debian package; together they make a turn-key academic security lab.

Who it's for

Who benefits from VITOS

👩‍💻

Students

Hands-on cybersecurity learning with safety rails that prevent costly mistakes.

  • Boot to a fully-loaded Kali workstation with every standard pentest tool, no setup
  • Experiment freely inside a Firejail sandbox — no risk of damaging shared infra
  • Work through 8 structured lab exercises (recon, exploit, forensics, malware, wireless, web, mobile, capstone)
  • See instantly if a command is "out of scope" for the current exercise
  • Get academic-integrity protection: AI flags are advisory and reviewed by humans, never automatic punishment
🔬

Researchers & Faculty

A research-grade platform with proper isolation and audit trails for publishable work.

  • Authorized Ghost Mode with WireGuard/Tor netns + nftables kill-switch for legitimate red-team research
  • Real-time admin dashboard: see every student session, terminal, network flow, and AI rationale
  • Local-only AI engine: no student data leaves the lab, FERPA/DPDP compliant out of the box
  • Reproducible builds via live-build + Docker — same ISO every time, signed and checksummed
  • One-click PDF incident reports for academic-integrity proceedings
  • Pen-test report template ready for IRB / ethics committee review

⌘ Reassemble + verify

3 commands · zero magic
  1. Download every .part*.iso file plus SHA256SUMS from the release page
  2. Concatenate the parts in order:
# reassemble cat vitos-v1-*-amd64.part*.iso > vitos-v1-amd64.iso # verify sha256sum -c SHA256SUMS # burn to USB (replace sdX) sudo dd if=vitos-v1-amd64.iso of=/dev/sdX bs=4M status=progress oflag=sync

Or boot it directly in QEMU/KVM:

qemu-system-x86_64 -enable-kvm -m 6144 -smp 4 \ -cdrom vitos-v1-amd64.iso -boot d \ -vga virtio -display gtk

Default credentials (forced password change on first login): admin:changeme, student:changeme. Replaced by FreeIPA on a successful realm join.
